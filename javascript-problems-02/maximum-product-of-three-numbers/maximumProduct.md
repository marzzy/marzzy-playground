# Maximum Product of Three Numbers

## General Information

- Run Environment:
  - Node
- Problem Source:
  - Leetcode
- Problem Difficulty
  - Easy
- Source Link:
  - [ID: 628](https://leetcode.com/problems/maximum-product-of-three-numbers/)

## Details

### Description

Given an integer array `nums`, find three numbers whose product is maximum and return the maximum product.
&nbsp;

### Examples

> **`01`**
>>**Input :**
>>
>>      nums = [1,2,3]
>>
>>**Output :**
>>
>>      6
>>
> **`02`**
>>**Input :**
>>
>>      nums = [1,2,3,4]
>>
>>**Output :**
>>
>>      24
>>
>

&nbsp;

### Constraints

- `3 <= nums.length <= 104`
- `-1000 <= nums[i] <= 1000`
